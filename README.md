## About this proyect

The application allows visitors to users can upload, share and view videos that other users upload to the platform.
-Create user interface that allows users to view, share and once registered allows uploading videos

- Connect with API rest, which allows the correct functionality of the page
- Creation of API rest, with all the endpoints for creation, modification of users and everything necessary to upload videos

## How to run locally

Clone the project, install node_modules (using npm install)
Create local file .env.local (copy .env.example), and put your api key

```bash
npm run dev
```

The app will be in http://localhost:3000
